---
title: "30. Клонування репозиторіїв"
---

### Цілі

- Навчитися робити копії репозиторіїв.

Якщо ви працюєте в команді, наступні 8 уроків досить важливі в розумінні, тому що ви майже завжди будете працювати з клонованими репозиторіями.

## _01_ Перейдіть в директорію `repositories`

<Run>
```
cd ..
pwd
ls
```
</Run>

<p className="highlight">Зараз ми знаходимося в директорії `repositories`.</p>

<Output>
```git
==include[output/30-01.txt]
```
</Output>

У цей момент ви повинні знаходитися в директорії `repositories`. Тут має бути єдиний репозиторій під назвою `work`.

## _02_ Створіть клон репозиторія `work`

Давайте створимо клон репозиторія.

<Run>
```
git clone work home
ls
```
</Run>

<Output>
```git
==include[output/30-02.txt]
```
</Output>

У вашому списку репозиторіїв тепер має бути два репозиторія: оригінальний репозиторій `work` і клонований репозиторій `home`.
