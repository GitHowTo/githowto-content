### Цілі

- Навчитися підтягувати зміни зі спільного репозиторія.

Швидко перемкніться в репозиторій `home` і підтягніть зміни, щойно відправлені в спільний репозиторій.

<Run>
```
cd ../home
```
</Run>

<p className="highlight">Зараз ми знаходимося у репозиторії `home`.</p>

Продовжіть з...

<Run>
```
git remote add shared ../work.git
git branch --track shared main
git pull shared main
cat README
```
</Run>
