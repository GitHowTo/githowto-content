title: "39. Додавання віддаленого репозиторія"
---

### Цілі

- Додати чистий репозиторій як віддалений до нашого оригінального репозиторія.

Додаймо репозиторій `work.git` до нашого оригінального репозиторія.

<Run>
```
cd work
git remote add shared ../work.git
```
</Run>

<p className="highlight">Зараз ми знаходимося в репозиторії `work`.</p>
