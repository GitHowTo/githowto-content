### Цілі

- Дізнатися про гілки у віддалених репозиторіях.

## _01_ Подивіться на клонований репозиторій

Давайте поглянемо на клонований репозиторій.

<Run>
```
cd home
ls
```
</Run>

<Output>
```git
==include[output/31-01.txt]
```
</Output>

Ви побачите список всіх файлів на верхньому рівні оригінального репозиторія (`README`, `index.html` і `css`).

## _02_ Перегляньте історію репозиторія

<Run>
```
git log --all
```
</Run>

<Output>
```git
==include[output/31-02.txt]
```
</Output>

Ви побачите список всіх комітів у новий репозиторій, і він повинен збігатися з історією комітів в оригінальному репозиторії. Єдина різниця повинна бути в назвах гілок.

## _03_ Віддалені гілки

Ви побачите гілку `main` (`HEAD`) в списку історії. Ви також побачите гілки з дивними іменами (`origin/main`, `origin/style` і `origin/HEAD`). Ми поговоримо про них трохи пізніше.
