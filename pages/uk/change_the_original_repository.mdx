---
title: "34. Зміна оригінального репозиторія"
---

### Цілі

- Внести деякі зміни в оригінальний репозиторій, щоб потім спробувати підтягнути та злити зміни з віддаленої гілки в поточну.

## _01_ Внесіть зміни в оригінальний репозиторій `work`

<Run>
```
cd ../work
```
</Run>

<p className="highlight">Зараз ми знаходимося в репозиторії `work`.</p>

Внесіть наступні зміни у файл `README`:

<File file="README">
```
This is the Hello World example from the Git tutorial.
(changed in origin)
```
</File>

Тепер додайте цю зміну і зробіть коміт

<Run>
```
git add README
git commit -m "Changed README in original repo"
```
</Run>

## _02_ Далі

Тепер в оригінальному репозиторії є більш пізні зміни, яких немає в клонованій версії. Далі ми винесемо і зіллємо ці зміни до клонованого репозиторія.
