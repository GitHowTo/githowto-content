### Цілі

- Навчитися зливати підтягнуті зміни в поточну гілку.

## _01_ Злийте підтягнуті зміни в локальну гілку `main`

<Run>
```
git merge origin/main
```
</Run>

## _02_ Ще раз перевірте файл `README`

Зараз ми повинні побачити зміни.

<Run>
```
cat README
```
</Run>

<Output>
```git
==include[output/36-02.txt]
```
</Output>

Ось і зміни. Хоча команда `git fetch` не зливає зміни, ми можемо вручну злити зміни з віддаленого репозиторія.

## _03_ Команда `pull` (підтягнути)

Команда `fetch` дає вам повний контроль над тим, що саме підтягується та зливається в локальні гілки, але для зручності існує також команда `pull`, яка підтягує та зливає зміни з віддаленої гілки у вашу поточну гілку одним викликом.

```
git pull
```

...еквівалентна наступним двом діям:

```
git fetch
git merge origin/main
```
