---
title: "40. Відправка змін"
---

- Дізнатися, як передавати зміни у віддалений репозиторій.

Оскільки чисті репозиторії зазвичай розташовуються на якомусь віддаленому сервері, ви не зможете туди просто зайти, щоб підтягнути зміни. Тому нам необхідно якось передати наші зміни в репозиторій.

Почнемо зі створення зміни, яку потрібно передати в репозиторій. Відредагуйте `README` і зробіть коміт змін:

<File file="README">
```
This is the Hello World example from the Git tutorial.
(changed in the origin and pushed to shared)
```
</File>

<Run>
```
git switch main
git add README
git commit -m "Added shared comment to readme"
```
</Run>

Тепер надішліть зміни до спільного репозиторія.

<Run>
```
git push shared main
```
</Run>

_Спільним_ називається репозиторій, що одержує відправлені нами зміни. Пам'ятаєте, ми додали його в якості віддаленого репозиторія в попередньому уроці?
