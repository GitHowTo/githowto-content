title: "37. Додавання гілки відстеження"
---

### Цілі

- Навчитися додавати локальну гілку, котра відстежує зміни віддаленої гілки.

Гілки, котрі починаються з `remotes/origin` є гілками оригінального репозиторія. Зверніть увагу, що у вас більш немає гілки з назвою `style`, але Git знає, що в оригінальному репозиторії гілка `style` була.

## _01_ Додайте локальну гілку, котра відстежує віддалену гілку

<Run>
```
git branch --track style origin/style
git branch -a
git log --max-count=2
```
</Run>

<Output>
```git
==include[output/37-01.txt]
```
</Output>

Тепер ми можемо бачити гілку `style` у списку гілок і у лозі.
