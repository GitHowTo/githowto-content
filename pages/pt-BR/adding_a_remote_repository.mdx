---
title: "39. Adicionando um repositório remoto"
---

### Metas

- Adicionar um repositório bare como um remoto para o nosso repositório original.

Vamos adicionar o repositório `work.git` ao nosso repositório original.

<Run>
```
cd work
git remote add shared ../work.git
```
</Run>

<p className="highlight">Agora nós estamos no repositório `work`.</p>
