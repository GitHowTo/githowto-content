## _01_ Remoção da tag `oops`

A tag `oops` cumpriu sua função, vamos removê-la. Isso permitirá que o mecanismo interno do Git remova os commits residuais, que agora não são referenciados por nenhum branch ou tag.

<Run>
```
git tag -d oops
git log --all
```
</Run>
